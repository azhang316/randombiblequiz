   # Random Bible Quiz

   A simple quiz to help you memorize the order of books in the Bible — so you can navigate a physical Bible faster.

   ## Why?

   Instead of flipping back and forth aimlessly (bogosort-style) to find a book, knowing what comes before and after each book lets you do a binary search starting from the middle. Much faster.
